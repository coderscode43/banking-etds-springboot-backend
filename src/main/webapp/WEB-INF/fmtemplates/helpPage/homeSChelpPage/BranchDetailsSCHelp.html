RO Details

  • User can enter the required data in the search fields to filter out the data .
  • After entering and selecting the details from the dropdown and clicking on the search button the user will get the filtered out data inside the table.
  • After clicking on Add button User will redirect to new page where User called Add Branch Details. and User can add new branch Details.
  • Export to excel button creates an excel of the table shown on the web site.
    User can generate an excel after filtering the data and an excel with the filtered out data will be downloaded in the users system.
  • User can see all data and searched data inside this table .
  • If the table has any data available the User can see all the details of that particular data just by double clicking on a particular row.
  • Previous Button redirects the user to the previous page and shows the previous 100 row inside the table.
  • Next Button redirects the page to the Next page and shows the next 100 row inside the table.
  • Go Button redirects the page to the page number entered inside the page number input box.
    User can manually put the page number and it will directly redirect the page to the respective page number.
