Add Branch Details Explanation

    • All fields with red star are mandatory to be filled.
  1. User enter ro code in RO Code fields.
  2. User enter branch name in Branch Name fields.
  3. User enter branch contact no in Branch Contact No fields.
  4. User enter branch pincode in Branch Pincode fields.
  5. User enter branch code in Branch Code fields.
  6. User enter branch email code in Branch Email fields.
  7. User enter branch address in Branch Address fields.
  8. User select branch state using Branch State Dropdown.
  9. After entering all the data click on save button to save Branch Details
  10. click to back button to go back to Branch Details Page
