Add RO details Explanation

    • All fields with red star are mandatory to be filled.
  1. User enter ro code in RO Code fields.
  2. User select ro state using RO State Dropdown.
  3. User enter branch name in Branch Name fields.
  4. User enter branch contact no in Branch Contact No fields.
  5. User enter ro pin code in RO Pin code fields.
  6. User enter ro name in RO Name fields.
  7. User enter ro email code in RO Email fields.
  8. User enter ro address in RO Address fields.
  9. After entering all the data click on save button to save Add RO details
  10. click to back button to go back to RO Details Page
