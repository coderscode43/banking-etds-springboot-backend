Correction Details Explanation

The Correction Details Page displays all the details that were entered by the user while adding a correction request like Ticket Number, Type of Correction,etc.
  • On clicking on the Add Response button, a pop-up box appears like the one shown below and the user has to fill in the response and he/she can also add a supporting document.

  • On clicking on the Checker Approved button, a pop-up box appears like the one shown below and the user has to fill in the reason for approval and he/she can also add a supporting document.

  • On clicking on the Reject button, a pop-up box appears like the one shown below and the user has to fill in the reason for the rejection and he/she can also add a supporting document.

  • On clicking on the Back button, the user will be redirected back to the Correction Request Details Page.

  • On clicking on the Download button, a PDF will be downloaded in to the users system.
