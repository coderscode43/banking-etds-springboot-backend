Upload Certificate Explanation

  • User can enter the required data in the search fields to filter out the data .
  • After entering and selecting the details from the dropdown and clicking on the search button the user will get the filtered out data inside the table.
  • After clicking on the upload certificate button a pop-up modal opens so that the user can fill all the required fields and upload a certificate by clicking on the add button.
  1. User can select a tan number from a dropdown.
  2. User can select a certificate type from a dropdown.
  3. User can select a financial year from a dropdown.
  4. User can select a quarter from a dropdown.
  5. User can upload a file.
  6. On clicking on add button User can upload a certificate with the details.
  7. User can close a modal by clicking on close button.
  • Clicking the Generate Zip button triggers a pop-up modal, enabling the user to fill in all necessary fields. Upon completion, they can generate a zip file by clicking the respective button within the modal.
