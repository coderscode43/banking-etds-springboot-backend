Download Certificate Explanation


Download Certificate For Single PAN

    • All fields with red star are mandatory to be filled.
  1. User has to enter pan number in PAN fields.
  2. User has to select tan number in TAN fields Dropdown.
  3. User has to select Type of Certificate using Type of Certificate Dropdown.
  4. User has to select year using year Financial Year Dropdown.
  5. User has to select quarter using Quarter Dropdown.
  6. User has to click on the download button to Download Certificate For Single PAN (Form 16A/16/27D) .

Download Certificate For Multiple PAN

    • All fields with red star are mandatory to be filled.
  1. User has to select tan number in TAN fields Dropdown.
  2. User has to select Type of Certificate using Type of Certificate Dropdown.
  3. User has to select year using year Financial Year Dropdown.
  4. User has to select quarter using Quarter Dropdown.
  5. User has to enter multiple pan number in PAN fields and check how to enter pan number using i mark.
  6. User has to click on the download button to Download Certificate For Multiple PAN (Form 16A/16/27D) .

Download All Certificates

    • All fields marked with a red star are mandatory to be filled.
  1. User has to select tan number in TAN field dropdown.
  2. User has to select Type of Certificate using Type of Certificate dropdown.
  3. User has to select year using year Financial Year dropdown.
  4. User select quarter using Quarter Dropdown.
  5. User has to click on the download button to Download All Certificates (Form 16A/16/27D) .
